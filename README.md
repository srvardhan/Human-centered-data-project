# Human-Centered Data Annotation & Prompt Optimization

## Overview
A freelance-style project focused on data annotation, prompt refinement, and AI response evaluation to improve output quality and workflow efficiency.

## Objective
Enhance AI response quality through structured human review, prompt optimization, and performance evaluation.

## Project Scope
- Data Annotation
- Prompt Engineering
- AI Response Evaluation
- Reporting & Analysis in Excel

## Dataset
| Message | Label |
|---|---|
| My internet is not working since morning | Complaint |
| How can I change my plan? | Inquiry |
| Thank you for resolving the issue quickly | Positive Feedback |
| The server is down, please fix ASAP | Urgent Issue |

## Prompt Engineering Example
**Basic Prompt:** Write an email for leave  

**Optimized Prompt:** Write a professional leave request email to my manager for one day leave due to medical reasons

## Outcome
Demonstrated practical experience in human-centered data workflows, AI quality review, analytical thinking, and problem-solving.
